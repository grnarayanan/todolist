Daily To-Do List

Welcome to the daily to-do list!

This module simulates the functionality of daily list of tasks. 
The user can add tasks, mark them as complete, and receive a daily summary of tasks completed.
Tasks clear each day to give you a fresh start. 

Usage Instructions:

Type 'a' to add a task. Then proceed to enter the name of the task. 
Type 'c' to mark a task as complete. Type the numeric position of the task on the list. 
Type 'e' to exit the to do list. 

Essentials Folder:

Contains a timestamp file for list reset and a file to store contents of the list.
